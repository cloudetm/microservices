# The Linux Command Line

https://www.safaribooksonline.com/library/view/the-linux-command/9781593273897/

note: commands are running on Ubuntu

## centos

```
$ cd vagrant/centos
$ vagrant up
$ vagrant ssh # user=vagrant password=vagrant
```

> run as root

```
[vagrant@localhost ~]$ su root
Password: vagrant
```
