# Analysis of "Songs for kids" in AngularJS

The project directory contains following files are shared between the host and guest (/vagrant folder).

```
- songs-angularjs-app
  - README.md
  - Vagantfile
  - web
    - css
    - data
    - index.html
    - js
    - lib
    - partials
```

Host: `web/` directory = Guest `/vagrant/web` folder
