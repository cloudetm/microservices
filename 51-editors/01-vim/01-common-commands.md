# Common Commands

## comment and uncomment

```
:18,28s/^/#
:18,28s/^#/
```

## delete all lines

```
:1,$d
```

## Search and replace

http://www.linfo.org/vi/search.html

> replace one `lemon` with `orange`

```
:s/lemon/orange/
```

> replace all `lemon` with `orange`

```
:%s/lemon/orange/
```
